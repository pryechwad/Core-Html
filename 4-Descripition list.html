Tech stack web development

HTML
HTML (Hypertext Markup Language): For structuring the content of web pages.
CSS
CSS (Cascading Style Sheets): For styling and layout of web pages.
Java Script
JavaScript: For adding interactivity and functionality to the user interface.
MongoDB
MongoDB is a NoSQL database that stores data in a flexible, JSON-like format called BSON (Binary JSON)
React
React is a JavaScript library for building user interfaces. It is maintained by Facebook and is widely used for creating dynamic and interactive front-end interfaces. React allows for the development of reusable UI components and provides a virtual DOM for optimizing performance.
Node js
Node.js is a runtime environment that allows you to execute JavaScript on the server side. It's built on Chrome's V8 JavaScript engine and provides an event-driven, non-blocking I/O model. Node.js is used as the server technology in the MERN stack.
